Calculator
==============

Command Line Interface Calculator

## Set up
```bash
cd calculator
npm install
```

## Usage
```js
./calc.js input
```

## Inputs
Only one argument allowed
- Number - (Number) ex: 123
- Expression - (String) ex: 'add 1 1'


## Output
Calculated result
